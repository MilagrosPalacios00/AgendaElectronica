#Agenda Electrónica en Java
#Descripción
Este proyecto es una aplicación de agenda electrónica desarrollada en Java. Permite a los usuarios gestionar contactos, incluyendo la entrada de nombres, apellidos, teléfonos y DNI. La aplicación valida los datos ingresados para asegurar que sean correctos y proporciona una interfaz gráfica intuitiva para interactuar con los datos.

#Funcionalidades
Validación de Datos: Verifica que los nombres y apellidos contengan solo letras, que los teléfonos tengan 10 dígitos y que los DNI tengan 8 dígitos.
Gestión de Registros: Permite a los usuarios guardar, visualizar, editar y navegar entre los registros de contacto.
Navegación Intuitiva: Facilita la navegación entre registros con botones para avanzar y retroceder.
Manejo de Errores: Muestra mensajes de error claros si los datos ingresados no son válidos.
#Tecnologías
#Lenguaje: Java
#Bibliotecas: Swing para la interfaz gráfica
